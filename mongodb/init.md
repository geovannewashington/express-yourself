# An initial overview on some concepts

## What is whitelistening on databases?

Whitelisting an IP address on a database means
to add your specific IP address to a security list, allowing it to connect to the database while
automatically denying all other IP addresses. This is a security measure that restricts access to
only authorized locations, preventing unauthorized connections and potential threats

## What is Cluster?

A cluster is a group of interconnected servers (nodes) that work together as a single system.

## A litte bit on mongoose

For the JavaScript environment, the official library is the `MongoDB Node.js Driver`
`mongoose` is not the official library developed by MongoDB for Node.js

`mongoose` sis on top of the official MongoDB Nodejs Driver, it provides a higher-level abstraction.
offering features such as: schema definition.

## Some Terms:

The database itself is a collection of namespaces.
A namespace is a collection of collections.
A collection is is a collection of documents.
A document is a collection of fiels, like JSON
